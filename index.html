Client vs Server

(Voltar)

A diferença entre client (cliente) e server (servidor) está na função que cada um desempenha em uma rede. O cliente é o dispositivo ou software que faz solicitações de serviços ou recursos, como acessar um site ou enviar um e-mail. Já o servidor é o sistema que responde a essas solicitações, fornecendo os dados ou executando operações conforme requisitado pelo cliente. Em outras palavras, o cliente inicia a comunicação, e o servidor processa as solicitações e envia de volta as respostas.

HTML

(Voltar)

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas da web. Ele define a estrutura básica de um website, permitindo a inclusão de texto, imagens, links, vídeos e outros elementos. Com HTML, é possível formatar o conteúdo, criar títulos, parágrafos, listas, e inserir links para navegar entre páginas. É uma linguagem de marcação que utiliza "tags" para delinear diferentes partes do conteúdo e dizer ao navegador como elas devem ser exibidasTML

Tags

(Voltar)

Tags são elementos fundamentais do HTML que definem a estrutura e o conteúdo de uma página da web. Elas são usadas para identificar diferentes partes do conteúdo, como títulos, parágrafos, imagens, links, e listas, dizendo ao navegador como exibir cada parte. Uma tag normalmente consiste em um nome de elemento cercado por colchetes angulares, como p para um parágrafo ou h1 para um título. As tags geralmente vêm em pares, uma de abertura (por exemplo, p) e uma de fechamento (por exemplo, /p), que envolvem o conteúdo que elas afetam. Algumas tags, como img , são auto-fechadas e não têm uma tag de fechamento correspondente.

Atributos

(Voltar)

Atributos em HTML são propriedades adicionais que fornecem informações extras sobre elementos e modificam o comportamento ou a aparência de uma tag. Eles são inseridos dentro da tag de abertura e geralmente consistem em um nome e um valor, separados por um sinal de igual. Por exemplo, o atributo src em uma tag 'img' especifica o caminho para uma imagem (img src="imagem.jpg"), enquanto o atributo href em uma tag 'a' define o destino de um link (a href="https://exemplo.com"). Atributos ajudam a personalizar e controlar como elementos da página são exibidos e interagem com o usuário.
